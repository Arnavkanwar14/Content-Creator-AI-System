Content Creator Crew

Automated AI crew that researches topics, writes scripts, and generates voice-over ready audio assets using Google Gemini.

View code See sample outputs

What it does

  • Researches a topic using a multi-agent CrewAI workflow.
  • Drafts structured reports and presentation-ready scripts.
  • Generates WAV narration using Gemini text-to-speech.
  • Saves everything inside the outputs/ folder.

How to run it locally

  1. Install Python 3.10+ and clone this repo.
  2. Copy env_template.txt → .env and add your Gemini API key.
  3. Run pip install -e . to install dependencies.
  4. Execute python -m content_creator.main to produce new assets.
